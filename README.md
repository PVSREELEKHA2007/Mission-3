# Mission 3 - React Portfolio Website

## 📌 Project Description

This project was developed as part of the VertexMind Web Development Internship.

The application is built using React and Vite. It demonstrates reusable React components, client-side routing using React Router, and state management using the useState hook.

## ✨ Features

- Responsive Navigation Bar
- Reusable Components
- Home Page
- About Page
- Project Detail Page
- State Management using useState
- Fast Development with Vite
- Responsive Design

## 📥 Installation

Clone the repository:
git clone https://github.com/PVSREELEKHA2007/Mission-3.git

Install dependencies: npm install

Run the project:
npm run dev

Open: http://localhost:5173
